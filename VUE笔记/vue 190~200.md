# 使用注意事项
- 明确提问:
    确保问题表述清晰明确，关键字的准确度，决定了A|对于需求的理解。
- 细化需求:
    将问题拆分成多个小问题，可以帮助AI更具针对性地回答，也利于即时纠错。
- 添加背景信息:
    提供有关问题背景的详细信息，也可以给AI预设一个角色，将有助于AI生成更具深度和价值的回答。
- 适当引导:
    比如：“例如”、“请注意”、“请使用“等，来告诉模型你期望它做什么或者不做什么
- 限制范围：
    通过限定回答的范围和长度，可以引导AI生成更精炼的回答
